int tag_qualify_symbol_name(_str &qualified_name, _str search_name, _str context_class, _str context_file, _str(&tag_files)[], boolean case_sensitive, ...)

Qualify the given class symbol by searching for symbols with its name in the current context/scope. This is used to resolve partial class names, often found in class inheritance specifications. The current object must be an editor control or current buffer.

Parameters:

qualified_name - (output) "qualified" symbol name

qualified_max - number of bytes allocated to qualified_name

search_name - name of symbol to search for

context_class - current class context (class name)

context_file - current file name

tag_files - list of tag files to search

case_sensitive - case sensitive tag search?

visited - (optional) hash table of prior results

depth - (optional) depth of recursive search

Returns: int

qualified name if successful, 'search_name' on error.

Category:

Tagging Functions
