int VSAPI SETagSetClassParents(int dbHandle, const slickedit::SEString &class_name, const slickedit::SEString &class_parents)

Specify what classes and interfaces a given file derives from. This will create an entry for 'class_name' if it does not already exist.

Fully qualified class names are preferred, but un-qualified class names may be used also, however, they may not be resolved correctly if there are duplicates with the same name.

Parameters:

dbHandle - database handle

class_name - Fully qualified class name

class_parents - a semi-colon seperated list of class names.

Returns: int

0 on success, <0 on error.

Category:

Tagging Functions
