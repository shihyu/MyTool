int VSAPI SETagInsertFileEnd(int dbHandle, slickedit::SEHashSet&oldTags)

Clean up after inserting a series of tags from a single file for update. Doing this allows the tag database engine to remove any tags from the database that are no longer valid.

Parameters:

dbHandle - database handle

Returns: int

0 on success, <0 on error.

Category:

Tagging Functions
