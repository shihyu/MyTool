_command typeless untabify_region()

Used in EMACS emulation. Converts consecutive tab characters in the highlighted text to the equivalent number of space characters. If no highlighted text exists, an invisible mark is used which contains the last highlighted text not deleted or the last text pasted by the emacs_paste command.

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Methods, Editor Control Methods
