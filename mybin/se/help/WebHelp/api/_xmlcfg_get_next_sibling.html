int _xmlcfg_get_next_sibling(int iHandle, int NodeIndex, int NodeTypeFlags=VSXMLCFG_NODE_ELEMENT_START | VSXMLCFG_NODE_ELEMENT_START_END)

Gets the next sibling child index from the specified index. The root index is always TREE_ROOT_INDEX, so to traverse the tree start there.

Parameters:

iHandle - Handle of tree to get index from

NodeIndex - Index of node to get sibling node index for

NodeTypeFlags - Indicates node type of child to find. 1 or more VSXMLCFG_NODE_* flags. See _xmlcfg_add() for more information on this parameter.

Returns: int

Returns the node index of the next sibling of the node specified by NodeIndex. -1 is returned if the node specified has no next sibling.

Category:

XMLCFG Functions
