void _TreeGetComboDataCol(int col, STRARRAY comboBoxLines, ...)

Get combo box data for all items in a given column

Parameters:

col - column of item to set data for (0 is the leftmost)

comboBoxLines - Lines to be filled in when combo box is dropped

comboBoxStyle - One of the PSCBO_* styles. Defaults to PSCBO_NOEDIT

Applies To:

Tree View

Category:

Tree View Methods
