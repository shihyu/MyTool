void _TreeSetRowColor(int nodeIndex, int FGColor, int BGColor, int Flags)

Set color for a row in a tree control. Actually sets each cell in the row (as opposed to _TreeSetColColor)

Parameters:

int - nodeIndex Index to set the color for

int - FGColor RGB color for text in this row

int - BGColor RGB color for background in this row

int - Flags Combination of font flags. Currently only F_INHERIT_FG_COLOR, and F_INHERIT_BG_COLOR are supported. These will cause this cell to be colored the "default" color, either the default tree color, or one set by _TreeSetColColor

Applies To:

Tree View

Category:

Tree View Methods
