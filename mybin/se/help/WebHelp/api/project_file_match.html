_str project_file_match(_str name, boolean find_first)

Parameters:

name - file name prefix to match

find_first - 'true' to find first, 'false' to find next

Returns: _str

Returns next file name in the current project which is a prefix match of name. Returns '' when no more matches are found.

Categories:

Buffer Functions, Completion Functions

See Also:

f_match, dir_match, wkspace_file_match
