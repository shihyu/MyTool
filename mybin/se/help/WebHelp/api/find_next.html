_command int find_next(boolean doPrevious=false)

Ctrl+G or "Search", "Next Occurrence"

Repeats a search initiated by a search command in same direction and with the same search options.

Parameters:

doPrevious - If true, repeats search in the opposite direction.

Returns: int

Returns 0 if successful. Otherwise STRING_NOT_FOUND_RC is returned. On error, message is displayed.

Applies To:

Edit Window,Editor Control

Categories:

Editor Control Methods, Edit Window Methods, Search Functions

find, replace, gui_replace, gui_find, find_prev
