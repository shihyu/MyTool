int tag_current_context()

Return the index of the current context item.

For synchronization, threads should perform a tag_lock_context(false) prior to invoking this function.

Returns: int

Returns the index of the current context tag, calculated using the current buffer seek position. Returns <0 on error or 0 if no current context.

Category:

Tagging Functions
