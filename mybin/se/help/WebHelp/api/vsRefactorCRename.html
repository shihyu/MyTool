int VSAPI vsRefactorCRename(int handle, const char *filename, const char *symbolPath, seSeekPos startSeekPos, seSeekPos endSeekPos, const char *newName, int flags)

Rename a symbol

Parameters:

handle - Handle to transaction

filename - Filename to find symbol in

symbolName - Name of symbol to rename

startSeekPos - Seek position of symbol or position to start search

endSeekPos - Seek position to end symbol search (0 if begin is actual symbol)

newName - New name for symbol

flags - Optional flags

Returns: int

0 on success, <0 on error

Category:

Refactoring Functions
