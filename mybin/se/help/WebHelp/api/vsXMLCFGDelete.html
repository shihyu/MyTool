int VSAPI vsXMLCFGDelete(int iHandle, int NodeIndex, int OnlyDeleteChildren=a)

Delete a node and its children from an XML tree.

Parameters:

iHandle - Handle to an XML tree

NodeIndex - Index to delete

OnlyDeleteChildren - When true, only the children of the node specified are deleted.

Returns: int

Returns 0 if successful

Category:

XMLCFG Functions
