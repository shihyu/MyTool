int VSAPI vsLanguageInheritsFrom(const char *parent, const char *pszLangId=a)

Does the current source language match or inherit from the given language?

If 'lang' is not specified, the current object must be an editor control.

Parameters:

parent - language ID to compare to

pszLangId - current language ID (default=p_LangId)

Returns: int

'true' if the language matches, 'false' otherwise.

Since:

13.0

Attention:

This function is thread-safe provided pszLangId is not NULL or the empty string.

Applies To:

Edit Window, Editor Control

Category:

Tagging Functions
