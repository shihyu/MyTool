boolean getInsertBlankLineBetweenBeginEnd(_str langID, boolean defaultValue=null)

Retrieves the insert blank line between braces setting for the given language.

Parameters:

langID - language ID (see p_LangId)

defaultValue - (optional) the value to return if no value is saved for this setting in the given language. If this is not specified, then the application's default value for this language setting is returned

Returns: boolean

insert blank line between braces setting for language

Category:

LanguageSettings API
