int VSAPI vsUTF8IsLower(const char *p, const char *pend, int *plen)

Parameters:

p - Pointer to UTF-8 character

pend - Pointer to end of UTF-8 source buffer.

plen - Set to number of bytes in UTF-8 character read.

Returns: int

Returns non-zero value if the UTF-8 character is a lower case letter. Otherwise, 0 is returned.

Category:

Unicode Functions
