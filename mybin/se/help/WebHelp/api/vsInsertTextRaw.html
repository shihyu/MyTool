int VSAPI vsInsertTextRaw(int wid, const char *pBuf, int BufLen=-, int Binary=a, unsigned char NLCh1='\r', unsigned char NLCh2='\n')

This function is identical to the vsInsertText function, except that the input string is in the same format as the internal buffer data which can be SBCS/DBCS or UTF-8. See "Unicode and SBCS/DBCS C API Programming".
