int dbg_session_step_out(int session_id, int thread_id)

Step out of the current function and back into the calling function.

Parameters:

session_id - debugger session ID (1 .. max_session)

thread_id - Thread ID to step

Returns: int

0 on success, <0 on error.

Since:

10.0

Example:

   int thread_id = dbg_get_cur_thread();
   int status = dbg_session_step_out(session_id, thread_id);
   if (status) {
      return status;
   }

Category:

Debugger Functions
