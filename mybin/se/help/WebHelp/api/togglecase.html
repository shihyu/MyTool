_str togglecase(_str string, int utf8=-1)

Parameters:

string - Input string.

utf8 - -1 specifies that input and output string are UTF-8 if UTF-8 support is enabled. 0 specifies that input and output string are SBCS/DBCS. 1 specifies that input and output string are UTF-8.

Returns: _str

Returns string with case converted on all strings, lower to upper, and upper to lower.

Example:

   togglecase('ABc') == 'abC'
   togglecase('abc') == 'ABC'

Category:

String Functions

See Also:

lowcase, upcase, togglecase_selection
