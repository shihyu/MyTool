int pas_proc_search(_str &procname, int find_first)

If proc_name is '', then this procedure searches for the first or next occurrence of a PASCAL language function definition. Search always starts from the cursor position. The find_first parameter indicates whether this is the first or next call. proc_name is set to the name of the function found.

If proc_name is not '', then a search for a PASCAL language function definition with name proc_name is performed. Searching begins at the cursor position. find_first must be non-zero if proc_name is not ''. The cursor is placed on the function definition, if one is found.

Returns: int

Returns 0 if successful. Otherwise a non-zero value is returned.

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Methods, Editor Control Methods, Search Functions
