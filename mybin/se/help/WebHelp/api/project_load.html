_command void project_load(_str projectOptions='')

List files in the current project and lets you quickly edit one or more.

Parameters:

projectOptions - Specifies which project files to load files from. By default it lets you select from all files in your workspace. If you give it the "-p" option, it will restrict the list to your current active project. Finally, you can give a specific project file (or list of project files).
