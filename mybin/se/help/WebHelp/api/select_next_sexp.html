_command int select_next_sexp()

Select the next S-expression. If called repeatedly, extend the selection to the next S-expression.

Returns: int

Returns 0 if successful, <0 on error.

Applies To:

Edit Window, Editor Control

Categories:

CursorMovement Functions, Edit Window Methods, Editor Control Methods

See Also:

next_sexp, prev_sexp, select_prev_sexp, cut_next_sexp, cut_prev_sexp
