_command int next_tag(_str quiet="", _str flags="")

Places cursor on next tag definition, skipping any tags filtered out by the Defs tool window. If the quiet argument is given an is not "", no error message is displayed.

Returns: int

Returns 0 if successful. Otherwise, a non-zero value is returned.

Applies To:

Edit Window, Editor Control

Category:

Search Functions

See Also:

next_proc, prev_proc, prev_tag
