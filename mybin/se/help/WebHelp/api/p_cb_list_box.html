CTL_LISTBOX p_cb_list_box

Deprecated. Combo box is no longer split into 4 controls. Just use the regular p_window_id for everything

Returns the window id of a list box sub-control of a combo box. A combo box consists of 4 controls: the root window, text box, picture box, and list box. The properties and methods of the sub-controls may be accessed individually with the p_cb, p_cb_text_box, p_cb_picture, p_cb_list_box instance handle properties. The p_cb_list_box property is only available when the control is displayed.

Example:

defeventtab form1;
combo1.on_create()
{
   // Active the list box of the combo box control
   p_window_id=p_cb_list_box;
   for (i=1;i<=100;++i){
      _lbadd_item('line='i)
   }
   p_window_id=p_cb;
}

Applies To:

Combo Box, Drive List

Categories:

Combo Box Properties, Drive List Properties
