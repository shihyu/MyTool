seSeekPosRet VSAPI vsBufGetNofLines(int buf_id)

Returns the number of lines in a buffer including no save lines (lines with the VSLF_NOSAVE flag set).

Parameters:

buf_id - Buffer id returned by vsPropGetI(wid,VSP_BUF_ID), vsBufEdit, or vsBufMatch.

Returns: seSeekPosRet

Returns number of lines in buffer including no save lines (lines with the VSLF_NOSAVE flag set).
