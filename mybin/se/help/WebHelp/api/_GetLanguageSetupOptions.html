int _GetLanguageSetupOptions(_str lang, VS_LANGUAGE_SETUP_OPTIONS &setup)

Retrieve the basic options for the given language. This applies only to the options stored in the def-language-[lang] variable.

Parameters:

lang - Language ID, see p_LangId

setup - Struct containing all language setup options

Category:

Miscellaneous Functions
