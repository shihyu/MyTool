void VSAPI vsSelectEditMode(int wid, const char *pszLangId=a, int reserved=a)

Deprecated. Use vsSetEditorLanguage().

Sets the language specific information such as tabs, margins, color coding, etc. to the specified language's options.

Parameters:

wid - Window id of editor control. 0 specifies the current object.

pszLangId - Language ID. See p_LangId. For list of language types, use our Language Options dialog ("Tools", "Options","Language Setup...").

Category:

Edit Window Methods
