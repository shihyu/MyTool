const char*VSAPI vsFileRelative(char *pszDestFilename, const char *pszSourceFilename, const char *pszToDir=a, int AddDotDots=a)

Converts pszSrcFilename to a path specification which is relative to the directory specified by pszToDir.

Parameters:

pszDestFilename - Result of convert pszSrcFilename to a relative path specification.

pszSrcFilename - Filename with fully qualified path specification. This may NOT be a relative path specification.

pszToDir - Directory with or without trailing backslash (UNIX: forward slash). If 0 or "" is given, the current directory is used.

addDotDots - Indicates with ".." are allowed in the result filename.

Returns: const char*VSAPI

Returns pszDestFilename.

Category:

File Functions

See Also:

vsFileAbsolute, vsFileAbsoluteTo, vsFileRelative
