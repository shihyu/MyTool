void restore_search(_str search_string, int flags, _str word_re, _str ReservedMore=null, int flags2=0)

Restores the current search data. The built-in function save_search may be called to return the current search data which is used by the built-in functions repeat_search, _select_match, and match_length. Flags is a combinations of the search flags constants below (defined in "slick.sh"):

  • VSSEARCHFLAG_IGNORECASE
  • VSSEARCHFLAG_MARK
  • VSSEARCHFLAG_POSITIONONLASTCHAR
  • VSSEARCHFLAG_REVERSE
  • VSSEARCHFLAG_RE
  • VSSEARCHFLAG_WORD
  • VSSEARCHFLAG_UNIXRE
  • VSSEARCHFLAG_NO_MESSAGE
  • VSSEARCHFLAG_GO
  • VSSEARCHFLAG_INCREMENTAL
  • VSSEARCHFLAG_WRAP
  • VSSEARCHFLAG_HIDDEN_TEXT
  • VSSEARCHFLAG_SCROLL_STYLE
  • VSSEARCHFLAG_BINARY
  • VSSEARCHFLAG_BRIEFRE
  • VSSEARCHFLAG_PRESERVE_CASE
  • VSSEARCHFLAG_WORDPREFIX
  • VSSEARCHFLAG_WORDSUFFIX
  • VSSEARCHFLAG_WORDSTRICT
  • VSSEARCHFLAG_HIDDEN_TEXT_ONLY
  • VSSEARCHFLAG_NOSAVE_TEXT
  • VSSEARCHFLAG_NOSAVE_TEXT_ONLY
  • VSSEARCHFLAG_PROMPT_WRAP

Parameters:

flags2 - currently contains flags for color coding. Constants are not yet defined for this.

Category:

Search Functions
