int VSAPI vsRefactorCPullUpFindMembers(int handle, const char *filename, const char *symbolName, seSeekPos startSeekPos, seSeekPos endSeekPos, const char *destClassName, char *members, const char *dependencyFiles)

Find the members of the given class and check for dependencies.

Parameters:

handle - Handle to transaction

className - Name of class

filename - Name of file containing class.

startSeekPos - Seek position of beginning of class reference.

endSeekPos - Seek position of end of class reference.

destClassName - Name of class to move members to.

members - (out)String containing members of class along with dependency information.

dependencyFiles - Files that contain function body definitions.

Returns: int

0 on success, <0 on error members is in the form: number of members [@] memberIndex [@] memberDescription [@] memberType [@] number of dependents [@] dependent index [@] isADependency(true or false) [@] dependentDescription [@] ... ... dependent index will be -1 if it is a global dependency that cannot be moved.

Category:

Refactoring Functions
