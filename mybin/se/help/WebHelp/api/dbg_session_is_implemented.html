int dbg_session_is_implemented(int session_id, int capability)

Check if the given debugger session implements the given capability.

Parameters:

session_id - debugger session ID (1 .. max_session)

capability - debugger capability (DEBUG_CAPABILITY_*)

Returns: int

>0 if the feature is implemented, 0 if not, <0 on error.

Since:

10.0

Category:

Debugger Functions
