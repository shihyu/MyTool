_str getXMLWrapOptions(_str langID, _str defaultValue=null)

Gets the xml wrap options for this language. This value is available on the GUI on the Formatting form for each applicable language.

Parameters:

langID - language ID (see p_LangId)

defaultValue - if the value is not available, then we just return this value. If this parameter is not specified, then the default value for the overall def-var is used.

Returns: _str

xml wrap options for this language

Category:

LanguageSettings API
