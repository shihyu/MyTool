int VSAPI vsEndSelectCompare(int wid, int markid=-)

Parameters:

wid - Window id of editor control. 0 specifies the current object.

markid - Handle to a selection returned by vsAllocSelection or vsDuplicateSelection. A mark_id of -1 identifies the active selection.

Returns: int

0
Current line is on last line of selection.
>0
Current line is after last line of selection.
<0
Current line is before last line of selection.
-1
Text is not selected or selection is not in current buffer.

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Methods, Editor Control Methods, Selection Functions

See Also:

vsBeginSelectCompare
