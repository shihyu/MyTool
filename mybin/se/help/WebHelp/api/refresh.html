void refresh(_str option="A")

We recommend you call this function with no parameters unless this does not meet your needs. This function acts as a method by forcing immediate display update of the current object.

Parameters:

option - may be one of the following:

'A'
Update all editor controls. This does not flush paint messages.
'W'
Flush paint messages to the current window.
'F'
This options is ignored and is present only for backward compatibility.

Applies To:

All Window Objects
