boolean p_pointer_style

Determines the preferred style of formatting a pointer declaration for the current buffer. This property is available only when the control is displayed.

Example:

   p_pointer_style = 0;
      char *p;
   p_pointer_style = VS_C_OPTIONS_SPACE_AFTER_POINTER;
      char* p;
   p_pointer_style = VS_C_OPTIONS_SPACE_SURROUNDS_POINTER;
      char * p;

Applies To:

Editor Control, Edit Window

Categories:

Editor Control Properties, Edit Window Properties
