int dbg_session_restart(int session_id)

Restart the program running under the debugger and step into it. Resets all breakpoints. Note that this maintains the existing connections to the debugger and vsdebugio (where applicable).

Parameters:

session_id - debugger session ID (1 .. max_session)

Returns: int

0 on success, <0 on error.

Since:

10.0

Example:

To restart an active debug session. 

   int status = dbg_session_restart(session_id);
   if (status) {
      return status;
   }

Category:

Debugger Functions
