int old_list_buffers(_str option='', boolean hide_current=false)

Displays the old Select a Buffer dialog box. Displays selection list of all buffers being edited. The buttons on the right of this dialog box operate on the selected line in the buffer list box.

Parameters:

option - if this string contains "-h", hidden buffers will be shown if this string contains "-l", the dialog will perform as the "Link Window" dialog

hide_current - if true, hides the current buffer from the list

Returns: int

Returns 0 if switched buffers successfully.

Applies To:

Edit Window

Category:

Buffer Functions
