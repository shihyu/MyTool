VSDLLEXPORT VSINT64 vsXMLCFGGetAttributeInteger(int iHandle, int NodeIndex, VSPSZ pszAttrName, VSINT64 iDefaultValue=0)

Retrieves a numeric attribute value for the attribute name given. This function is thread-safe.

Parameters:

iHandle - Handle to an XMLCFG tree returned by vsXMLCFGOpen(), vsXMLCFGCreate(), or vsXMLCFGOpenFromBuffer().

NodeIndex - Tree node index.

pszAttrName - Name of attribute to return value of

iDefaultValue - default value to return if attribute not present

Returns: VSDLLEXPORT VSINT64

Returns attribute value if successful. Otherwise pszDevaultValue is returned.

Category:

XMLCFG Functions
