int tag_list_context_globals(int tree_wid, int tree_index, _str tag_name, boolean check_context, _str(&tag_files)[], int filter_flags, int context_flags, var vnum_matches, int max_matches, boolean exact_match, boolean case_sensitive, ...)

List the global symbols visible in the given list of tag files matching the given tag filters and context flags.

For synchronization, macros should perform a tag_lock_matches(true) prior to invoking this function.

Parameters:

tree_wid - Window ID of tree control to insert into, 0 indicates to insert into a match set, see tag_insert_match.

tree_index - Parent tree node index to insert item under, ignored if tree_wid is zero.

tag_name - Tag name or prefix of tag name to search for.

check_context - check for symbols in the current context?

tag_files - (Input) Tag files to search

filter_flags - Tag filter flags, only insert tags passing this filter. See tag_filter_type for more details.

context_flags - Context flags representing what to allow or restrict to. See tag_check_context_flags for more details.

vnum_matches - (Input, Output) Number of matches

max_matches - Maximum number of matches allowed

exact_match - Specifies exact match or prefix match (0)

case_sensitive - Specifies search case sensitivity.

visited - (optional) hash table of prior results

depth - (optional) depth of recursive search

Returns: int

Returns 0 on success, <0 on error.

Category:

Tagging Functions

See Also:

tag_insert_match, tag_tree_insert_tag, tag_tree_prepare_expand
