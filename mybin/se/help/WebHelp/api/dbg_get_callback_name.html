_str dbg_get_callback_name(int session_id)

Return the debugger system callback name for the given session ID

Parameters:

session_id - debugger session ID (1 .. max_session)

Returns: _str

"" on failure, otherwise return session callback name

Since:

10.0

Category:

Debugger Functions

See Also:

dbg_find_session, dbg_get_session_name
