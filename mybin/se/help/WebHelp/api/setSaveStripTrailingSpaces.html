void setSaveStripTrailingSpaces(_str langID, int value)

Sets whether files in this language are have their trailing spaces stripped when saved. This value is available on the File Options form for each applicable language.

Parameters:

langID - language ID (see p_LangId)

value - new value values:

                           STSO_OFF - do not strip spaces
                           STSO_STRIP_ALL - strip all trailing spaces
                           STSO_STRIP_MODIFIED - strip
                           spaces only from modified or
                           inserted lines

Category:

LanguageSettings API
