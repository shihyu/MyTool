int tag_check_for_template(_str tag_name, _str outer_class, boolean case_sensitive, _str(&tag_files)[], _str &template_sig)

Look up tag_name and see if it is a template class. If so, return the signature of the template in template_sig. The current object must be an editor control or current buffer.

For synchronization, threads should perform a tag_lock_context(false) prior to invoking this function.

Parameters:

tag_name - Tag name to check if it is a template class.

outer_class - Name of class that tag belongs to.

case_sensitive - Speciifies case_sensitive comparison

tag_files - (Input) Tag files to search

template_sig - (Output) returns signature of template_sig

Returns: int

1 if a match was found and sets template_sig, 0 of none found.

Category:

Tagging Functions
