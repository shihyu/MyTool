int _SearchQNofSkipped()

When the p_TruncateLength is non-zero, replace operations are skipped if the resulting line becomes longer than the truncation length. The _SearchInitSkipped function is used in conjunction with _SearchQNofSkipped and _SearchQSkipped to return information about what was skipped. Call this function before performing a search and replace operation to reset NofSkipped and Skipped lines data returned by the _SearchQNofSkipped and _SearchQSkipped functions.

Returns: int

Returns the number of replace operations that were skipped.

Applies To:

Edit Window
