_str dirnq_match(_str name, boolean find_first)

Parameters:

name - directory name to search for

find_first - 'true' to find first, 'false' to find next

Returns: _str

Returns next directory name which is a prefix match of name. find_first must be non-zero to initialize matching. Returns '' when no more matches are found.

"dirnq" means "directory no quote". This function does not quote the items returned.

Categories:

Completion Functions, File Functions
