int refactor_c_replace_literal(int handle, _str filename, _str literalName, _str constantName, int flags)

Replace a literal with a constant

Parameters:

handle - Handle to transaction

filename - Filename to find symbol in

literalName - Name of literal to convert to a constant

constantName - Name of constant

flags - Optional flags

Returns: int

0 on success, <0 on error

Refactoring Functions
