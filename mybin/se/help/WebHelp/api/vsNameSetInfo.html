int VSAPI vsNameSetInfo(int index, const char *pInfo, int InfoLen=-)

This function is used to set command function attributes or user defined additional name information.

Parameters:

index - Index into names table of item.

pInfo - New name information string.

InfoLen - Number of characters in pInfo. -1 means pInfo is null terminated.

Returns: int

Returns 0 if successful.

Attention:

This function is thread-safe.

Category:

Names Table Functions
