int refactor_c_encapsulate(int handle, _str filename, _str symbolName, _str getterName, _str setterName, _str methodName, int beginSeekPos, int endSeekPos, int formattingFlags, int syntaxIndent)

Encapsulate a field. Make a field private and make public getter and setter functions.

Parameters:

handle - Handle to transaction

filename - Filename to find symbol in

symbolName - Name of symbol to encapsulate

getterName - Name of getter function to create

setterName - Name of setter function to create #param methodName Name of method declaration to insert getter and setter after

beginSeekPos - Seek position of symbol or position to start search

endSeekPos - Seek position to end symbol search (0 if begin is actual symbol)

formattingFlags - formatting flags, bitset of VSREFACTOR_FORMAT_* (above)

syntaxIndent - Syntax indent amount

Returns: int

0 on success, <0 on error

Category:

Refactoring Functions
