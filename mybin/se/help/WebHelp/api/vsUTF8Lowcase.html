void VSAPI vsUTF8Lowcase(char *p, size_t buflen)

Converts UTF-8 text to lower case. Under Windows, this function relies on the WIN32 function CharLowerBuffW which only support characters below 65536. Characters above 65537 are not changed.

Parameters:

pBuf - Buffer to convert.

BufLen - Length of buffer to convert.

Category:

Unicode Functions
