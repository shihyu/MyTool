VSSTATIC int vsUTF8MemICmp(const char *pBuf1, const char *pBuf2, int len)

Case insensitive compare. Characters are compared by character value and not using any Unicode standard.

Parameters:

pBuf1 - Pointer to UTF-8 buffer 1

pBuf2 - Pointer to UTF-8 buffer 2

len - Number of bytes to compare

Returns: VSSTATIC int

0     Strings are equal
>0    pBuf1>pBuf2
<0    pBuf1

Category:

Unicode Functions
