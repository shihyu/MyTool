int _getProjectFiles(_str workspaceFile, _str projectFile, _str(&filelist)[], int absolutePath, int projectHandle=-1)

Retrieves a list of the files in the given project.

Parameters:

workspaceFile - full path to workspace file

projectFile - full path to project file

filelist - array to fill with file names

absolutePath - non-zero to retrieve absolute paths, zero to retrieve paths relative to the project file

Returns: int

int 0 on success, non-zero otherwise

Category:

Project Functions
