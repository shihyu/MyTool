int tag_find_included_by(_str file_name, _str &included_by)

Retreive the name of first the source file that included (directly or indirectly), the given file (expected to be an include file).

Parameters:

file_name - Full path of file that was included

included_by - (Output) full path of source file

Returns: int

0 on success, < 0 on error, BT_RECORD_NOT_FOUND_RC if not found.

Category:

Tagging Functions
