int VSAPI vsPropTest(int wid, int prop_id, int object_id=0)

Parameters:

wid - Window id of object or index to resource returned by vsFindIndex, vsNameMatch, VSP_CHILD, or VSP_NEXT. 0 specifies the current object.

prop_id - One of the VSP_??? constants in "vs.h".

object_id - One of the VSOI_??? constants in "vs.h" Ignored unless window_id==0

Returns: int

Test if the given property is safe to get for the given window ID without causing the interpreter to throw an error. Returns 0 if ok, <0 if not safe.

Example:

     int col;
       col=vsPropTest(0,VSP_COL);

Applies To:

All Window Objects

Category:

Miscellaneous Functions

See Also:

vsPropGetB, vsPropGetZ, vsPropSetZ, vsPropSetB, vsPropGetI, vsPropGetI64, vsPropSetI, vsPropSetI64, vsPropGetL, vsPropSetL
