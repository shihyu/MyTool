int _select_block(_str markid="", _str options='E')

Starts or extends the block selection specified. Used for processing columns of text. The first _select_block becomes the pivot point. Subsequent _select_block calls will select the area between the pivot point and the cursor.

Parameters:

mark_id - mark_id is a selection handle allocated by the _alloc_selection built-in. A mark_id of '' specifies the active selection or selection showing and is always allocated.

options - options is a string of zero or more of the following:

'C'
Specifies that the selection extend as the cursor moves.
'E'
(Default) Requires that _select_block be executed to end the text selection.
'P'
specifies a persistent select style. Macros use this to help determine if a selection should be unhighlighted when the cursor moves.

Returns: int

Returns 0 if successful. Possible returns are TEXT_NOT_SELECT_RC. and TEXT_ALREADY_SELECTED_RC.

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Methods, Editor Control Methods, Selection Functions

See Also:

_select_char, _select_line, _show_selection
