_command typeless delete_whole_word()

Delete current word at cursor.

Returns: typeless

Returns 0 if successful. Common return codes are TOO_MANY_SELECTIONS_RC, and STRING_NOT_FOUND_RC. On error, message is displayed.

Applies To:

Edit Window, Editor Control, Text Box, Combo Box

Categories:

Clipboard Functions, Combo Box Methods, Edit Window Methods, Editor Control Methods, Text Box Methods

See Also:

delete_word, cut_word
