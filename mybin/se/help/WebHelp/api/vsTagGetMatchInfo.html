int VSAPI vsTagGetMatchInfo(int match_id, char *pszTagFile, char *pszTagName, char *pszTypeName, int &tag_flags, char *pszFilename, int &start_line_no, int &start_seekpos, int &scope_line_no, int &scope_seekpos, int &end_line_no, int &end_seekpos, char *pszClassName=a, char *pszReturnType=a, char *pszArguments=a, char *pszExceptions=a, char *pszClassParents=a, char *pszTmplSignature=a)

Retrieve information about the given search match.

Parameters:

match_id - match ID to look up (from tag_insert_match)

pszTagName - (Output, required) Name of tag or symbol

pszTypeName - (Output, optional) Tag type name, see tag_get_type.

tag_flags - (Output) Tag flags indicating symbol attributes, VS_TAGFLAG_

pszFilename - (Output, optional) File name as stored in database.

start_line_no - (Output) Line number that tag occurs on in file_name.

start_seekpos - (Output) Seek position where the tag begin in file_name.

scope_line_no - (Output) Line number that tag scope begins, eg. function scope

scope_seekpos - (Output) Seek position where the tag scope begins.

end_line_no - (Output) Line number that tag ends on in file_name.

end_seekpos - (Output) Seek position where the tag ends in file_name.

pszClassName - (Output, optional) Name of class that tag belongs to.

pszReturnType - (Output, optional) Declared type/value of symbol or function

pszArguments - (Output, optional) Function/Macro arguments.

pszExceptions - (Output, optional) Exceptions thrown or other function details

pszClassParents - (Output, optional) Class parents

pszTmplSignature - (Output, optional) Template signature

Returns: int

0 on success, <0 on error.

Category:

Tagging Functions
