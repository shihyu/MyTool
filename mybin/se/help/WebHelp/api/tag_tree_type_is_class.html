int tag_tree_type_is_class(_str type_name)

Speed-demon way to determine if the given type is a class, struct, or union (class, struct, union).

Parameters:

type_name - String specifying tag's type name (from tag_get_info)

Returns: int

1 if the item is a class, 0, otherwise.

Category:

Tagging Functions
