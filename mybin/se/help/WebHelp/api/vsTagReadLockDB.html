int VSAPI vsTagReadLockDB(int dbHandle, int ms=0)

Lock the given database to prevent other threads from writing to it. This will obtain a read lock on the database, blocking writer threads, but permitting other read threads to continue to use the database.

Parameters:

dbHandle - database handle returned from vsTagOpenDB()

ms - millisecond timeout to wait before giving up on obtaining the reader lock.

Returns: int

0 on success, <0 on error

Category:

Tagging Functions

See Also:

vsTagOpenDB(), vsTagCloseDB(), vsTagWriteLockDB(), vsTagReadLockDB(), vsTagUnlockDB()
